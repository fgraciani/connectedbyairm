Route

Route.annotation

Properties Definition Type
multipleIdentifier A suffix for the route designator, indicating alternates for homonymous routes (especially for military training routes).
missing data A specified route designed for channelling the flow of traffic as necessary for the provision of air traffic services, from the end of the take-off and initial climb phase to the commencement of the approach and landing phase.
militaryTrainingType A code indicating the type of military training activity for which the route has been designed.
designatorNumber The number of the route designator.
designatorSecondLetter The letter of the route designator. Depending on the region, letters may have special significance with regard to the route usage limitations.
flightRule A code indicating the flight rules to be observed by aircraft. For example, VFR and/or IFR.
internationalUse A code indicating if the route segment is international or domestic.
locationDesignator A textual description of the area in which a designated route is situated.
designatorPrefix A prefix for the route designator, indicating whether the route is upper ATS, reserved for helicopters or reserved for supersonic flights.
militaryUse A code indicating whether the route is for exclusive use by military aircraft.
name A free text name by which the route is known.
type Route classification.
userOrganisation Indicates the Organisation normally originating activity on a military training route.
