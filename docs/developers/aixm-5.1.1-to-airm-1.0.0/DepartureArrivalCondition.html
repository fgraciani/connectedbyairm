DepartureArrivalCondition

DepartureArrivalCondition.annotation

Properties Definition Type
annotation
engineType type of aircraft engine allowed for the condition
missing data A condition which is established for a departure or an arrival
maximumCrossingAtEnd The uppermost vertical position at the end point, when flying on the route portion in the direction indicated in the RoutePortionUsage.
minimumCrossingAtEnd The lowermost vertical position at the end point, when flying on the route portion in the direction indicated in the RoutePortionUsage.
minimumCrossingAtEndReference The reference surface used for the value of the minimumCrossingAtEnd. For example, Mean Sea Level, Ground, standard pressure, etc..
minimumEnrouteAltitude Minimum en-route altitude (MEA). The altitude for an en-route segment that provides adequate reception of relevant navigation facilities and ATS communications, complies with the airspace structure and provides the required obstacle clearance.
maximumCrossingAtEndReference The reference surface used for the value of the maximumCrossingAtEnd. For example, Mean Sea Level, Ground, standard pressure, etc..
