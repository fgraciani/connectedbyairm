RadarEquipment.location

The graphical location of the component

Properties Definition Type
serialNumber Serial number of the equipment.
range The applicable range of the equipment to detect aircraft.
missing data Physical radar equipment like PAR, ARSR, ASR, SECRA, etc..
digital Indicates if the radar equipment is digital or analogue.
dualChannel Indicates whether the component has dual frequency.
magneticVariation The angular difference between True North and Magnetic North measured at a given position and date.
magneticVariationAccuracy Accuracy of the magnetic variation.
militaryUseOnly Indicates the equipment is for military use only.
movingTargetIndicator Indicates whether a Moving Target Indicator (MTI) is available for the component. The MTI function eliminates ground clutter.
name Textual name of the equipment.
dateMagneticVariation The date on which the magnetic variation had this value.
rangeAccuracy Accuracy of the applicable range of the equipment to detect aircraft.
specialAircraftOnly Indicates the equipment is to be only used for certain type of aircraft that are capable of using it. In some instances, the slope angle or procedure can only be flown by high performance aircraft.
specialUseOnly Indicates the equipment is for special use procedures only such as Helipads, etc.
standbyPower Indicates how the component is powered if the primary power fails.
contact The contact information of the group or individual associated with the Radar Equipment.
