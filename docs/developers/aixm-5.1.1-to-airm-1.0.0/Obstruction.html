Obstruction

Obstruction.annotation

Properties Definition Type
obstaclePlacement location of obstacle in relation to a given point.
annotation
theVerticalStructure Vertical assessment controlling obstacle.
adjustment Operational factors applied to obstacle clearance altitudes to produce the safest usable altitude.
slopePenetration A value calculated by dividing 102.0 (a constant) with the angle of the glidepath
missing data [FAA 8260.3] The highest airspace, obstacle or terrain relative to a prescribed plane within a specified area.</para><para> NOTE: In precision approach procedures where obstacles penetrate the approach surface, the controlling obstacle is the one which results in the requirement for the highest decision height (DH).
surfacePenetration A code indicating whether the controlling obstacle penetrates the assessment surfaces. Rule: if surfacePenetration is "NO", then slopePenetration will not have a value.
requiredClearance Required altitude adjustment above any obstacle or terrain within a defined safety buffer region. The safety buffer allows for errors in the air and in the accuracy of data such as obstacle accuracy. This together with other adjustments (human factor adjustments, lighting adjustments, altimeter adjustments, etc). give the minimum altitude.
minimumAltitude The value of the minimum obstacle clearance altitude.
controlling Indication whether it is a "controlling obstruction" for the associated ObstacleAssessmentArea.
closeIn Close-In Obstacles are those that would penetrate the 40:1 OCS and would require a climb gradient to 200 feet or less above the departure end of runway. So if an aircraft climbs at the minimum standard 200 feet per NM, "Close-In Obstacles" are those within 1 NM from the end of the runway.
