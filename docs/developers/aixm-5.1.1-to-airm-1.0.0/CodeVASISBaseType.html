Properties Definition Type
3B_AVASIS 3 bar AVASIS
3B_ATVASIS 3 bar ATVASIS.
VASIS Visual approach slope indicator system.
OTHER Other type of visual approach slope indicator.
TVASIS T-shaped VASIS

HAPI Helicopter approach path indicator.
ILU A number of identical light units.
LCVASI Low cost visual approach slope indicator: 3 sets of 4 white lights on 3 mounts usually on only one side of the runway.
