Take-off run available

Semantic Correspondence
AIRM Concept Definition
CodeDeclaredDistanceType@TORA The length of runway declared available and suitable for the ground run of an aeroplane taking off.

LDAH

CodeDeclaredDistanceBaseType.LDAH

LDA for Helicopters

Semantic Correspondence
AIRM Concept Definition
CodeDeclaredDistanceType@LDAH The length of the FATO plus any additional area declared available and suitable for helicopters to complete the landing manoeuvre from a defined height.

LDA

CodeDeclaredDistanceBaseType.LDA

Landing distance available

Semantic Correspondence
AIRM Concept Definition
CodeDeclaredDistanceType@LDA The length of runway which is declared available and suitable for the ground run of an aeroplane landing.

RTODAH

CodeDeclaredDistanceBaseType.RTODAH

Rejected TODA for Helicopters

Semantic Correspondence
AIRM Concept Definition
CodeDeclaredDistanceType@RTODAH The length of the FATO declared available and suitable for helicopters operated in performance class 1 to complete a rejected take-off.

TODA

Take-off distance available

Semantic Correspondence
AIRM Concept Definition
CodeDeclaredDistanceType@TODA The length of the take-off run available plus the length of the clearway, if provided.

TODAH

Take-Off Distance Available for Helicopters

Semantic Correspondence
AIRM Concept Definition
CodeDeclaredDistanceType@TODAH The length of the FATO plus the length of helicopter clearway (if provided) declared available and suitable for helicopters to complete the take-off.
