RouteSegment

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirspaceInfrastructure:RouteAndProcedure:RouteSegment

A route or portion of route usually flown without an intermediate stop.

Properties Definition Type
designatorSuffix A suffix for the route designator defined in accordance with ICAO Annex 11. CodeRouteDesignatorSuffixType
endEnRouteSegmentPoint The end en-route segment point for the route segment. InfrastructurePoint
evaluationArea The obstacle evaluation area for the route segment. ObstacleAssessmentArea
availability The operational availability of the route segment. RouteAvailability
minimumCrossingAtEnd The lowermost vertical position at the end point, when flying on the route portion in the direction indicated in the route portion usage. ValVerticalDistanceType
maximumCrossingAtEnd The uppermost vertical position at the end point, when flying on the route portion in the direction indicated in the route portion usage. ValVerticalDistanceType
pathType The type of segment path. CodeRouteSegmentPathType
magneticTrack The initial magnetic track. Angle
startEnRouteSegmentPoint The start en-route segment point for the route segment. InfrastructurePoint
lowerLimit The vertical position of the route segment floor. ValVerticalDistanceType
minimumEnrouteAltitude The altitude for an en-route segment that provides adequate reception of relevant navigation facilities and ATS communications, complies with the airspace structure and provides the required obstacle clearance. ValVerticalDistanceType
level A code indicating if the route segment is in the upper airspace, the lower airspace or both. CodeLevelType
minimumObstacleClearanceAltitude The minimum altitude for a defined segment of flight that provides the required obstacle clearance. ValVerticalDistanceType
widthRight The distance from the centreline of the route segment to the right edge, when considering the direction from the start point to the end point. ValDistanceType
reverseMagneticTrack The reverse initial magnetic track. Angle
upperLimit The vertical position of the route segment ceiling. ValVerticalDistanceType
turnDirection The turn direction, if any, at the end of the segment in order to continue on to the next segment of the same route. CodeDirectionTurnType
navigationType Classification of the route portion for navigational purposes. CodeRouteNavigationType
trueTrack The initial true track. Angle
hasSignalGap Indication of a signal gap in the segment. Boolean
reverseTrueTrack The reverse initial true track. Angle
length The length of the path. ValDistanceType
widthLeft The distance from the centreline of the route segment to the left edge, when considering the direction from the start point to the end point. ValDistanceType
requiredNavigationPerformance The required navigation performance for the route portion expressed in NM. Decimal
airway A control area or portion thereof established in the form of a corridor. Airspace
